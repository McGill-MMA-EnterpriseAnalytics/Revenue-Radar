# Dataset Repository

## Introduction
Welcome to the dataset repository! This repository is designed to store and manage datasets for our project.

## Dataset Files
Please save the dataset files in the following locations:

- `Data/test.csv`: This file contains the test dataset.
- `Data/train.csv`: This file contains the training dataset.

## Usage
To use the datasets in your project, simply access the respective files mentioned above.

## Contributing
If you have additional datasets that you would like to contribute, please follow these steps:
1. Fork this repository.
2. Add your dataset file(s) to the appropriate location.
3. Submit a pull request.
